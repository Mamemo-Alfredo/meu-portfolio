## 📖 Sobre o Projeto

Portfólio web desenvolvido para apresentar meus projetos, habilidades e experiências como desenvolvedor front-end. O projeto combina design moderno com código eficiente, oferecendo uma experiência única e memorável para os visitantes.

### ✨ Funcionalidades

- 🎨 **Design Responsivo** - Adapta-se perfeitamente a todos os dispositivos
- ⚡ **Alta Performance** - Otimizado com Vite e lazy loading
- ♿ **Acessibilidade** - Desenvolvido com foco em inclusão
- 🚀 **SEO Otimizado** - Meta tags e estrutura semântica
- 💻 **Animações Suaves** - Transições elegantes com Framer Motion

### 🛠️ Tecnologias Utilizadas

- **React.js 19** - Biblioteca para interfaces
- **TailwindCSS 3** - Framework CSS utilitário
- **JavaScript ES6+** - Lógica e interatividade
- **Vite** - Build tool e desenvolvimento rápido

## 🎯 Objetivo

Criar uma presença online profissional que demonstre minhas habilidades técnicas, meu estilo de desenvolvimento e minha capacidade de criar experiências digitais excepcionais.

## 📦 Instalação e Uso

### Pré-requisitos

- Node.js (versão 16 ou superior)
- npm ou yarn
